What is Roundabout JS?

Simply, it's a jquery plugin that creates a interactive, three dementional slide show.

Why did I choose this plungin?

  1. It's easy to use
  2. It has a lot of practical application.
  3. It just looks awesome!

Let's Look at the requirement for Roundabout js.

  • Jquery 1.2 or later
  • You can download a physical file of Jquery Here.
  • Roundabout.js file, which you can find Here.
  • CSS if customization is required
For the demo I will be using Event Drag and Event Drop add-ons as well, this can be found on the Roundabout js github page.

Lets take a closer look at Roundabout js.

API

Roundabouts API is very simple.
Once we have linked the proper files all we need to do is create a <ul> tag, like this! 
<ul class='pictures'>
                            
    <li><img src='pic1.png' /></li>
                            
    <li><img src='pic2.png' /></li>       
                            
    <li><img src='pic3.png' /></li> 
                            
    <li><img src='pic1.png' /></li>
                            
    <li><img src='pic2.png' /></li>       
                            
    <li><img src='pic3.png' /></li> 
                            
</ul>

Next we use CSS to make it look the way we want.

.pictures{
    list-style: none;
    top: 100px;
    margin: auto;
}

.roundabout-holder{
      height: 300px;
      width: 600px;
      left:50px;
   }
   
.roundabout-moveable-item {
      height:200px;
      width: 200px;
      cursor: pointer;
   }

I took away the styling on the ul tag and made sizing calls to roundabout.

After that we create our slide using this jquery API.

                            $(document).ready(function() {

                                $('.pictures').roundabout({
                                     enableDrag: true,
                                     autoplay: true,
                                     autoplayDuration: 2000,
                                     autoplayPauseOnHover: true
                                  });
                                });

Once we reference the correct ul tag, its done!
